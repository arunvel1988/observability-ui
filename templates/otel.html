OpenTelemetry (OTel) is an open-source observability framework for cloud-native software.

It provides APIs, SDKs, and tools to collect traces, metrics, and logs from applications and infrastructure.

OTel helps developers and operators understand system behavior, troubleshoot issues, and improve performance.
