📊 Observability Tools Overview

This page gives you a quick overview of essential tools for monitoring, logging, and tracing, along with installation links and use cases.

🚀 Core Tools

  • Prometheus: Metrics collection & alerting (metrics)
  • Grafana: Dashboards & visualizations for metrics/logs
  • Loki: Centralized logging system (logs)
  • Jaeger: Distributed tracing system (traces)
  • OpenTelemetry: Vendor-neutral instrumentation
  • Elasticsearch + Kibana: Log analytics and dashboards
